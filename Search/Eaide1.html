Searching the directory to find EPFL people

This form is used for searching the LDAP directory to find EPFL people The search box displayed on the page can only be used to search for a name . If you want to search using other criteria, click on the link : "Power search" or "Advanced search".

You can use this search form to search in various fields at the same time, but at least 1 must be specified. The maximum number of results is limited to 200, if a search retrieves more than 200 results, an error will be generated.

You can use the character "*" to indicate any string of characters.
(e.g. 'le*eur' will find Lecommandeur, Leballeur and Le Meur).

Correcting an error in the directory

What should I do if something is wrong with the CSO directory?

No matter what problem you find in the directory (error in the information, poor presentation etc.), send a message regarding the problem to bottin@epfl.ch. Your e-mail will be forwarded to the correct person.

Dialling an EPFL telephone number

Telephone numbers are given with only five figures in the pages of the directory.

  • The area code for epfl is: 21 69
  • The country code for Switzerland is: 41

So if you want to phone the number 31234:

  • from EPFL: dial 31234
  • from UNIL: dial 9 31234
  • In Switzerland: dial 021 69 31234
  • from anywhere else : dial + 41 21 69 31234 (The + sign must be replaced with your international code)
  • if you encounter problems, call a local operator.
