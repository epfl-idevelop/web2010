Le Centre de Langues de l’EPFL géré par la Migros vous propose pendant les semestres des cours dans les langues suivantes: Français - Allemand - Anglais - Italien - Espagnol - Japonais - Chinois.

Pour les étudiants les cours sont gratuits pour toutes les langues, sauf pour le chinois, l’espagnol, et le japonais (9.-/l’heure).
